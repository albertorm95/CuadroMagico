Cuadro Mágico

-----------------------------------------

Descripción:

Un algoritmo que se encarga de crear un cuadro magico de multiples tamaños, un cuadro magico es un cuadro horizontal, vertical y diagonal seran la misma.

-----------------------------------------

Como funciona:

Se inicia solicitando el tamaño del Cuadro Magico
